# PANG 게임 PRD — Mission 1

## 1. 게임 개요

PANG은 전 세계를 무대로 튀어오르는 풍선을 모두 터뜨려 스테이지를 클리어하는 아케이드 게임이다.
Mission 1의 배경은 일본 후지산이며, Morning → Afternoon → Night 순서의 3개 스테이지로 구성된다.

---

## 2. 핵심 게임플레이

| 항목 | 내용 |
|------|------|
| 시점 | 2D 사이드뷰 |
| 플레이어 수 | 1인 |
| 조작 | 좌우 이동 + 위 방향 발사 |
| 클리어 조건 | 스테이지 내 모든 풍선 제거 |
| 실패 조건 | 풍선에 닿으면 목숨 차감 |

---

## 3. 풍선 메커니즘

풍선은 4단계 크기로 구분되며, 맞을 때마다 2개로 분열된다. 가장 작은 풍선은 맞으면 소멸한다.

```
Large → Medium (×2) → Small (×4) → Tiny (×8) → 소멸
```

- 크기가 클수록 바운스 높이가 높고 득점이 낮다.
- 크기가 작을수록 움직임이 빠르고 예측이 어려워 위험하다.

---

## 4. 무기 체계

| 무기 | 설명 |
|------|------|
| 기본 작살 (Harpoon) | 위로 발사, 풍선·블록 파괴 후 소멸 |
| 더블 작살 | 2발 동시 발사 |
| 파워 작살 | 천장에 닿을 때까지 유지, 지속 파괴 |
| 발칸 미사일 | 연사 가능, 블록은 파괴 불가 |

---

## 5. 아이템

| 아이템 | 효과 |
|--------|------|
| 시계 | 풍선 일시 정지 |
| 별 | 화면 내 모든 풍선 즉시 제거 |
| 모래시계 | 풍선 이동 속도 감소 |

| 다이너마이트 | 모든 풍선을 최소 크기로 축소 |
| 보너스 과일 | 추가 점수 획득 |

---

## 6. 스테이지 구성 요소

- **파괴 가능 블록**: 작살로 제거 가능, 풍선 이동 경로에 영향
- **파괴 불가 블록**: 장애물로 고정 배치
- **사다리**: 풍선 및 플레이어 이동에 영향
