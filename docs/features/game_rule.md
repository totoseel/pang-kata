# 게임 룰

## 기본 규칙

- 스테이지에 배치된 모든 풍선을 제거하면 클리어
- 풍선에 닿으면 목숨 1개 차감
- 목숨이 0이 되면 게임 오버

## 플레이어

- 화면 하단 바닥 위를 좌우로만 이동
- 위 방향으로 작살을 발사해 풍선을 터뜨림
- 동시에 발사 가능한 작살은 무기에 따라 다름

## 풍선 메커니즘

풍선은 4단계로 분열한다.

```
Large → Medium (×2) → Small (×4) → Tiny (×8) → 소멸
```

| 크기 | 바운스 높이 | 이동 속도 | 점수 |
|------|------------|----------|------|
| Large | 높음 | 느림 | 100점 |
| Medium | 중간 | 보통 | 200점 |
| Small | 낮음 | 빠름 | 300점 |
| Tiny | 매우 낮음 | 매우 빠름 | 400점 |

- 분열된 풍선은 좌우로 퍼지며 계속 바운스
- 분열할수록 화면에 풍선 수가 늘어 난이도 상승

## 무기

| 무기 | 설명 | 획득 방법 |
|------|------|----------|
| 기본 작살 | 위로 1발 발사, 풍선·블록 파괴 후 소멸 | 기본 지급 |
| 더블 작살 | 좌우 2발 동시 발사 | 아이템 획득 |
| 파워 작살 | 천장에 고정되어 일정 시간 유지 | 아이템 획득 |
| 발칸 미사일 | 연사 가능, 블록 파괴 불가 | 아이템 획득 |

- 기본 작살은 1발이 화면에 있는 동안 추가 발사 불가
- 무기 아이템을 획득하면 기존 무기가 교체됨

## 아이템

풍선을 터뜨리면 확률적으로 아이템이 드롭된다.

| 아이템 | 효과 | 지속 시간 |
|--------|------|----------|
| 시계 | 모든 풍선 일시 정지 | 일정 시간 |
| 별 | 화면 내 모든 풍선 즉시 제거 | 즉시 |
| 모래시계 | 풍선 이동 속도 감소 | 일정 시간 |
| 실드 | 피격 1회 무효화 | 1회 |
| 다이너마이트 | 모든 풍선을 Tiny 크기로 축소 | 즉시 |
| 보너스 과일 | 추가 점수 | 즉시 |

## 스테이지 요소

- **파괴 가능 블록**: 작살 충돌 시 제거, 풍선 이동 경로 변화
- **파괴 불가 블록**: 고정 장애물, 풍선이 튕겨 나감

## 점수 및 진행

- 모든 풍선 제거 시 스테이지 클리어 보너스 10,000점
- 클리어 후 다음 스테이지로 자동 전환
- 목숨은 스테이지 클리어 시 이월됨
