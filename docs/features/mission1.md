# Mission 1 — 후지산 (Mount Fuji)

## 개요

| 항목 | 내용 |
|------|------|
| 배경 | 일본 후지산 |
| 스테이지 수 | 3 (Morning / Afternoon / Night) |
| 난이도 | 입문 |
| 등장 풍선 크기 | Large, Medium, Small, Tiny |

---

## 스테이지 구성

### Stage 1 — Morning
- **풍선**: Large 1개
- **블록**: 없음
- **특징**: 장애물 없이 평탄한 바닥. 풍선 분열 패턴에 익숙해지는 튜토리얼 성격

### Stage 2 — Afternoon
- **풍선**: Large 1개, Medium 1개
- **블록**: 파괴 가능 블록 일부 배치
- **특징**: 동시에 여러 풍선이 존재하는 상황 대처 연습

### Stage 3 — Night
- **풍선**: Large 2개
- **블록**: 파괴 가능 블록 + 파괴 불가 블록 혼합
- **특징**: 블록 배치로 인해 풍선 이동 경로가 복잡해짐. Mission 1의 최종 관문

---

## 난이도 설계

### 풍선 수 증가 패턴

| 스테이지 | 초기 풍선 수 | 최대 동시 풍선 수 (완전 분열 시) |
|----------|------------|-------------------------------|
| Stage 1 (Morning) | 1 | 8 (Large 1개 완전 분열) |
| Stage 2 (Afternoon) | 2 | 12 (Large 8 + Medium 4) |
| Stage 3 (Night) | 2 | 16 (Large 2개 완전 분열) |

### 위협 요소 증가 흐름

```
Stage 1: 단순 분열 패턴 학습
    ↓
Stage 2: 다중 풍선 동시 처리
    ↓
Stage 3: 블록 장애물 + 다중 풍선 복합 대응
```

## 클리어 조건

- 각 스테이지의 모든 풍선(Tiny 포함)을 제거
- Stage 3 클리어 시 Mission 1 완료, Mission 2로 진행

## 주요 공략 포인트

- Large 풍선은 분열 직후 좌우로 튀므로 발사 타이밍이 중요
- Tiny 풍선은 빠르고 낮게 움직여 회피가 어려움 — 먼저 처리 우선순위 고려
- Stage 3에서는 파괴 불가 블록 사이에 끼인 풍선을 노릴 것
- 시계·별 아이템은 풍선이 많이 분열된 후 사용하면 효과적
