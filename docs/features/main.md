# 메인화면 구성

## 화면 레이아웃

```
┌─────────────────────────────────┐
│                                 │
│           PANG                  │  ← 게임 타이틀
│                                 │
│       [ GAME START ]            │  ← 시작 버튼
│                                 │
│       [ HIGH SCORE ]            │  ← 최고 점수 확인
│                                 │
└─────────────────────────────────┘
```

## 구성 요소

### 타이틀
- 게임 로고 "PANG" 표시
- 배경은 후지산 이미지 (Mission 1 배경)

### 메뉴
| 항목 | 동작 |
|------|------|
| GAME START | 게임 시작, Mission 1 Stage 1로 진입 |
| HIGH SCORE | 최고 점수 목록 표시 |

### 하이스코어 표시
- 현재 세션의 최고 점수를 화면 상단에 항상 표시

## 전환 흐름

```
메인화면 → GAME START 선택 → Mission 1 Stage 1 (Morning)
메인화면 → HIGH SCORE 선택 → 점수판 화면 → 메인화면으로 복귀
게임 종료(목숨 소진) → 메인화면으로 복귀
```
